- Version: 0.0.0.1
- Source: https://github.com/utrecht/agendas

Referentieimplementatie van de instemmingen Component (TC).

## Inleiding

Deze referentieimplementatie toont aan dat de API specificatie voor de instemmingen implementeerbaar is, en vormt een voorbeeld voor andere implementaties indien ergens twijfel bestaat.

Deze component heeft ook een [testomgeving](http://instemmingen.demo.zaakonline.nl/) waar leveranciers tegenaan kunnen
testen.
